# SATVIFY Frontend

SATVIFY is a modern health-food and subscription-based meal delivery platform built with React, TypeScript, and Vite. The application provides a seamless user experience for browsing healthy meals, managing subscriptions, placing orders, and tracking deliveries.

## Features

### Customer Side

* Responsive homepage with wellness-focused branding
* Dynamic menu with category filtering and search functionality
* Product details page with nutritional information
* Shopping cart and checkout flow
* User authentication pages (Login & Signup)
* User dashboard for profile, orders, and subscriptions
* Order history and order tracking interface
* Referral & rewards system UI
* Subscription management pages

### Admin Panel

* Admin dashboard
* Product management
* Order management
* Customer management
* Analytics dashboard

## Tech Stack

* React.js
* TypeScript
* Vite
* React Router DOM
* Context API (Cart Management)
* CSS3

## Project Goals

The project focuses on delivering a clean, responsive, and user-friendly experience across desktop, tablet, and mobile devices while showcasing modern frontend development practices.
